# Проект: Место

### Обзор

* Описание
* Технологии
* Основный блоки

**Описание**
Место - проект социальной сети, где люди могут выкладывать фотографии из своих путешествий

**Технологии**
Использованы HTML, CSS, JS


**Основные блоки**

* Шапка с заголовком
* Профиль с аватаркой
* Добавление фото
* Список фото
* Добавление в избранное
